Company Registration System:

A responsive registration system for companies, built using HTML, Tailwind CSS, Animate CSS, and JavaScript. It features form validation, real-time data fetching, and a loading spinner.

---

Project Overview:

This registration system allows companies to:
- Enter details like Company Name, Registration Number, Email, Phone, Address, Business Type, and Password.
- Fetch company details from an external API using the registration number.
- Experience real-time form validation and error messages.

---

Setup Instructions:

- You can open `index.html` by using vscode's Live Server extension

---
Features Implemented:

- Responsive Registration Form
- Real-Time Data Fetching from JSONPlaceholder API
- Loading Spinner** during data fetch
- Form Validation:
  - Email format validation
  - Password strength check
  - Confirm password match check
- Component-Based Structure for maintainable and reusable code

---

Technologies Used:

- HTML5: Semantic markup
- Tailwind CSS: Utility-first responsive design
- JavaScript (ES6+): Dynamic behavior and validation
- JSONPlaceholder API: Mock API for fetching company details
